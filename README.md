# 📝 AI Text Summarizer App

An intelligent text summarization web app built using **T5 Transformer model** and **Streamlit**. This application takes long text input and generates a concise, meaningful summary in seconds.

---

## 🚀 Features

* 🔹 Summarizes long text into short, readable content
* 🔹 Built using state-of-the-art **T5 Transformer model**
* 🔹 Simple and interactive UI with Streamlit
* 🔹 Fast inference with optimized settings
* 🔹 Clean and beginner-friendly project structure

---

## 🛠️ Tech Stack

* **Python**
* **Streamlit**
* **Hugging Face Transformers**
* **PyTorch**

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Student-ChinmayaMohanty/Text_summerizer_app.git
cd Text_summerizer_app
```

### 2️⃣ Create virtual environment

```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the App

```bash
streamlit run app.py
```

---

## 💡 How It Works

1. User inputs long text
2. The T5 model processes the text
3. Generates a concise summary
4. Output displayed instantly on UI

---

## ⚡ Optimization Tips

* Uses **t5-small** model for faster performance
* Model is loaded dynamically (not stored in repo)
* Can be further optimized using caching

---

## 🚫 Note

Model files are not included in this repository due to size limitations.
They are automatically downloaded from Hugging Face when the app runs.
